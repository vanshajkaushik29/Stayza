🏠 Stayza – Full Stack Booking Platform

Stayza is a modern full-stack accommodation booking platform inspired by real-world rental systems like Airbnb. It allows users to browse properties, view details, and make bookings with a smooth and secure experience. The project is built with a scalable MVC architecture and focuses on clean backend design, authentication, and media handling.

🚀 Live Demo

👉 https://stayza-qp2h.onrender.com/listings

📌 Features
🏡 Browse and view property listings
🔐 Secure user authentication (Signup/Login)
👤 Role-based access control (User/Admin)
📅 Booking system for properties
⭐ Review and rating system
🖼️ Image upload support using Cloudinary
📦 RESTful API architecture
💾 Persistent data storage with MongoDB
⚡ Fully responsive UI for all devices
